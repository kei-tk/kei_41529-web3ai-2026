# 棚 — Tana 🗄️

> 散らかった Downloads を、**AIが中身を読んで自動で棚に戻す**整理ダッシュボード。
> web3・AI概論 2026 / VPC「ファイル整理」のプロトタイプ **v2**。

第3回の VPC で選んだバグ **「ファイル整理」** をそのまま形にしたプロダクト。
「DLした瞬間にAIが中身を読んで自動でフォルダ分類」+「日付・科目・キーワードで絞れるダッシュボード」を実装した。

---

## 何ができる?

| VPC の要素 | 棚(Tana)での実装 |
|---|---|
| 😣 Pain: Downloadsに数百個並んでスクロール地獄 | カテゴリ(棚)ごとに自動グルーピング表示 |
| 😣 Pain: 検索しても出てこない | 名前 **+ 中身 + タグ** を横断する全文検索 |
| 😣 Pain: 思い出すヒントがファイル名しかない | AIが付ける**一言サマリ** + 中身プレビュー |
| 🎯 Gain: テーマや日付で探せる | 科目フィルタ / 期間フィルタ / タグクラウド |
| 🎯 Gain: DLした瞬間に自動分類 | ドロップ → AIが中身を読んで棚へ自動振り分け |
| 🎯 Gain: 中身プレビューで一覧 | カードのサマリ + クリックで中身プレビューパネル |

### 主な機能
- **AI自動分類**: ファイル名・拡張子・(テキストなら)中身を読み、9つの棚(講義資料/課題/レポート/コード/画像/メディア/領収書/圧縮/その他)へ自動仕分け
- **科目の自動推定**: web3・AI概論 / 数学 / プログラミング / 英語 …
- **横断検索**: 名前だけでなく中身・サマリ・タグも対象
- **フィルタ**: 棚・科目・期間(今週/今月/それ以前)・タグ
- **中身プレビュー** + **AI確信度**表示
- **ドラッグ&ドロップ**で自分の実ファイルを投入可能(テキスト系は中身も読む)

> ⚠️ プロトタイプのため、分類エンジンは本物の LLM ではなく**キーワード辞書ベース**で「中身を読んで意味で仕分ける」体験を再現している(`src/lib/classify.ts`)。将来 Gemini 等の API に差し替え可能な構造。

## 動かし方

```bash
cd tana
npm install
npm run dev      # → http://localhost:5173
```

ビルド: `npm run build` → `dist/` を Vercel 等にデプロイ。

「🗂️ 散らかったDLを読み込む」を押すとデモデータ(散らかった26ファイル)が読み込まれ、AIが棚分けする様子が見られる。

## 技術スタック
- Vite + React 18 + TypeScript
- Tailwind CSS v4
- 状態管理は React の `useState` / `useMemo` のみ(外部ライブラリなし)

## ディレクトリ
```
src/
  App.tsx              # 全体の状態・レイアウト
  types.ts             # OrgFile モデル
  lib/
    classify.ts        # ★ AI分類エンジン(中身を読んで仕分け)
    categories.ts      # 棚(カテゴリ)定義
    demoData.ts        # 散らかったDownloadsのサンプル
    ingest.ts          # 取り込み(デモ/実ファイル)
    format.ts          # 表示フォーマッタ
  components/          # Sidebar / FileCard / PreviewPanel / IngestOverlay / Logo
```
